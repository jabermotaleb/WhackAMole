# 🔨 Whack-a-Mole

A browser-based Whack-a-Mole game built with ASP.NET Core 9 MVC, Bootstrap 5, and JavaScript.

## How to Play
- Click **Start Game**
- Click the moles before they disappear
- You have 30 seconds — get as many as you can!

## Tech Stack
- ASP.NET Core 9 MVC
- Bootstrap 5
- JavaScript
- C#

## Run Locally

```bash
cd WhackAMole
dotnet run
```

Open: **http://localhost:5000**
